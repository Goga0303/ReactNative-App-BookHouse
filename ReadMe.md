#BookHouse
Book House är en app med Google books api hjälper användaren att utorska Googles bokhylla. Man kan som användare utforska Google books utbud och spara böcker i sina "Favoriter". 

#Funktioner 
- Det finns en sökfilter som man kan ta hjälp av vid sökandet av en bok. 
- Man kan spara böcker i sina favoriter. 
- Man med få titel och beskriving av en bok uppläst till en.
